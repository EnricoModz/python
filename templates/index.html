🐍 O que é Python?

Python é uma linguagem poderosa, versátil e fácil de aprender, muito usada em diversas áreas como web, automação, IA e ciência de dados.

  • ✅ Sintaxe limpa e objetiva
  • ✅ Muita documentação e comunidade
  • ✅ Frameworks incríveis (Flask, Django...)
  • ✅ Ideal para iniciantes e profissionais

🔧 O que é POO?

POO (Programação Orientada a Objetos) permite modelar o mundo real usando código, organizando programas em classes e objetos.

  • 🧱 Classes e Objetos
  • 🔁 Herança
  • 🔐 Encapsulamento
  • 🌀 Polimorfismo

📘 Exemplo: Fatorial

def fatorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * fatorial(n - 1)

print(fatorial(5))  # 120

📗 Exemplo: Número Primo

def eh_primo(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True

print(eh_primo(7))  # True

📙 Exemplo: POO

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def falar(self):
        print(f"Olá! Meu nome é {self.nome} e tenho {self.idade} anos.")

p1 = Pessoa("João", 25)
p1.falar()
